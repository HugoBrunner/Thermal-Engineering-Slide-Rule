# Thermal Engineering Slide Rule
This project was created mainly for fun.

I have always found slide rules fascinating, and I wanted to design one that highlights my own field of engineering: thermal engineering. I chose the Dittus-Boelter correlation because it has been in use since 1930 and is still widely used today. It is also particularly well suited to a slide rule, since it relies only on multiplication and exponentiation.

$$Nu = 0.023 \cdot Re^{0.8} \cdot Pr^{n}$$

With $n=0.3$ in cooling and $n=0.4$ in heating.

In practice, this kind of tool would not be very useful for daily engineering work, as such calculations are now typically handled by software, scripts, or spreadsheets. Still, it is an interesting way to explore both the history of engineering tools and the fundamentals of heat transfer correlations.

This GitHub repository contains the Python code used to generate SVG slide rules for calculating the Nusselt number from the Dittus–Boelter correlation. It also includes examples of slide rules generated with the code.
